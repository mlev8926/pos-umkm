Informasi & Panduan Aplikasi
Alur Kerja Kasir
  1. Pilih Produk — Klik kartu produk di grid kiri untuk menambahkan ke keranjang.
  2. Atur Jumlah — Gunakan tombol (+) dan (-) di keranjang untuk mengatur kuantitas.
  3. Diskon — Terapkan diskon per item (klik ikon tag di item keranjang) atau diskon global (% atau nominal).
  4. Bayar — Klik shortcut nominal atau ketik manual. Tombol UANG PAS mengisi otomatis = total setelah diskon.
  5. Konfirmasi — Tekan Bayar & Simpan. Kembalian akan ditampilkan.
  6. Cetak Struk — Struk otomatis muncul. Klik Cetak Struk untuk cetak ke printer termal 58mm.
Jenis Diskon
  • Diskon Per Item — Klik ikon tag pada item di keranjang. Bisa berupa persen (%) atau nominal (Rp).
  • Diskon Global — Diskon keseluruhan yang diterapkan ke total belanja. Bisa berupa persen (%) atau nominal (Rp).
Catatan Penting
  • Semua data tersimpan di localStorage browser — data tetap ada walau browser ditutup.
  • Jangan hapus cache browser jika ingin menyimpan data transaksi.
  • Gunakan Export CSV di tab Laporan untuk backup data penjualan.
